# OpenCode Mission Control

OpenCode plugin for parallel AI coding sessions in isolated git worktrees.

## Installation

```bash
npm install opencode-mission-control
```

## Usage

```typescript
import { plugin } from 'opencode-mission-control';

// Plugin implementation
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Test
bun run test
```
